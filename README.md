## Overview
Eunoia is a simple web-based journaling application designed to help users record and reflect on their daily thoughts.  
The project was built to practice core front-end development concepts and demonstrate the ability to apply programming logic in a practical context.

## Features
- Clean and minimal user interface
- Structured navigation between pages
- Journal-style layout intended for personal reflection
- Designed as a foundation for adding interactive features

## Technologies Used
- HTML5
- CSS3
- JavaScript (basic)

## Purpose of the Project
This project was created as a learning exercise to:
- Strengthen my understanding of web development fundamentals
- Practice building and structuring a multi-page website
- Prepare for further development involving user input handling and data storage

## What I Learned
Through this project, I learned:
- How to structure a website using semantic HTML
- How to style layouts using CSS for readability and consistency
- How to organize files and deploy a website using GitHub Pages
- The importance of planning functionality before implementation

## Future Improvements
- Add a journal entry input form
- Implement data storage using JavaScript localStorage
- Display saved entries dynamically
- Improve accessibility and responsiveness
